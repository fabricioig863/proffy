## 🚀 Projeto 

Projeto desenvolvido na trilha discovery ministrada pelo Mayk brito da Rockteseat no evento Next Level Week 02 
com o objetivo de homenagear os profissionais da educação. A aplicação desenvolvida tem como objetivo unir professor e aluno, quem deseja ensinar, com quem deseja aprender.

HTML
CSS
Javascript
Node js
Express js
Sqlite

## Como-executar ?

```bash
# Clonar o repositório
git clone https://github.com/fabricioig863/proffy.git

# Entrar no diretório
cd proffy

# Baixar as dependências
npm install express
npm install sqlite
npm install nodemon
npm install nunjucks

# Executar o servidor
npm run dev
```
Após os passos acessar o endereço no navegador `http://localhost:5500/`
